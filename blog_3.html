Analyzing Diabetes Trends in India Using Python

Diabetes is a growing health concern in India, affecting millions of people. To understand its impact better, I recently analyzed a small dataset related to diabetes in India using Python. I used Pandas to filter the data and Matplotlib to visualize trends, specifically focusing on gender distribution.

Filtering Data with Pandas

started by loading the dataset into a Pandas DataFrame and filtering it based on gender. The dataset included three categories:

  • Male
  • Female
  • Others
Using Pandas, I extracted relevant data to see how diabetes cases were distributed among these groups.

Visualizing Data with Matplotlib

To get a clearer picture, I used Matplotlib to create a pie chart showing the percentage of diabetes cases for each gender.

From the analysis, the diabetes prevalence among different gender groups is:

  • Male: 33.1%
  • Female: 34.4%
  • Other: 32.6%

These percentages show that diabetes is fairly evenly distributed across all gender groups, with females having the highest prevalence (34.4%). This could indicate potential biological, lifestyle, or healthcare-related factors that make females slightly more susceptible.

However, the differences are not very large, suggesting that diabetes affects all genders almost equally.
